# NodeJS MySQL Upload Images
>Upload And Save To MySQL
### Installing

1. Clone The Project
```
git clone https://github.com/SharpTH/crop-image-before-uploading.git
```
2. Install dependencies
```
cd crop-image-before-uploading
npm install
```
